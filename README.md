# 프로젝트 개요
* 개발 기간 : 2025.08.21 ~ 2025.08.27
* 개발 인원 : 5인
* 핵심역할 : 좌석 예매 시스템
* 주요 특징
    * MVC 패턴 기반의 깔끔한 소스 구조 (Controller, DAO, DTO 분리)
    * 좌석 예매, 음식/굿즈 주문, 경기 일정 관리 기능을 통합한 올인원 플랫폼
    * 사용자 권한(User/Admin)에 따른 차별화된 대시보드 제공
# 기술 스택
Backend
Java 17

Spring Boot

Spring Data JPA

H2 Database (또는 MySQL 설정 지원)

Maven

Frontend
HTML5, CSS3, JavaScript

OpenLayers (지도 라이브러리)

Chart.js (데이터 시각화)

Thymeleaf (템플릿 엔진)
