# 프로젝트 개요
* 개발 기간 : 2025.08.21 ~ 2025.08.27
* 개발 인원 : 5인
* 핵심역할 : 좌석 예매 시스템
* 주요 특징
    * MVC 패턴 기반의 깔끔한 소스 구조 (Controller, DAO, DTO 분리)
    * 좌석 예매, 음식/굿즈 주문, 경기 일정 관리 기능을 통합한 올인원 플랫폼
    * 사용자 권한(User/Admin)에 따른 차별화된 대시보드 제공
# Tech Stack
* Backend
    * Java 17
    * Spring Boot
    * Spring Data JPA
    * H2 Database (또는 MySQL 설정 지원)
    * Maven
* Frontend
    * HTML5
    * CSS3
    * JavaScript
    * OpenLayers (지도 라이브러리)
    * Chart.js (데이터 시각화)
    * Thymeleaf (템플릿 엔진)
# 주요 기능
1. 지도 기반 식당 조회
OpenLayers API를 활용한 지도 인터페이스 제공.

학교 및 주변 식당의 위치를 마커로 표시.

마커 클릭 시 식당의 상세 정보(메뉴, 위치 등) 확인 가능.

2. 식사 기록 관리 (Lunch Log)
학생들이 이용한 식당과 메뉴를 기록.

식당별 방문 횟수 및 메뉴 데이터를 데이터베이스에 저장.

3. 데이터 시각화 및 통계 (Chart)
Chart.js를 활용하여 식사 데이터를 다양한 차트로 제공.

식당별 선호도, 메뉴 분포 등을 시각적으로 분석.

관리자용 전체 통계 및 개별 식당 통계 페이지 지원.

4. 식당 및 메뉴 관리
식당 정보(이름, 좌표, 연락처 등) 등록 및 수정.

식당별 메뉴 리스트 관리 기능.
